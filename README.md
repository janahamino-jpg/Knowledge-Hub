# Knowledge Hub Website

Welcome to **Knowledge Hub**, a learning platform dedicated to technology, telecommunications, personal development, and global innovations.

This repository contains the HTML, CSS, and media files for the website.

---

## Features

- Responsive layout with semantic HTML5 structure
- Navigation menu with links to articles, videos, and resources
- Latest articles with images and educational content
- Embedded educational videos
- Downloadable resources (PDFs)
- Social media links (LinkedIn, Instagram)
- Footer with credits and social media prompts
